Bu çalışmada, konuşma sinyallerinden elde edilen akustik özellikler kullanılarak konuşmacının duygusal durumunun otomatik olarak sınıflandırılması amaçlanmaktadır. 
Problem, denetimli öğrenme kapsamında ele alınan çok sınıflı bir sınıflandırma problemidir.
Ele alınan duygu sınıfları; nötr, mutlu, öfkeli, üzgün ve şaşkın olarak belirlenmiştir. 
Çalışmanın temel hedefi, sınırlı süre içerisinde temel bir model geliştirerek başlangıç performansının elde edilmesidir.
